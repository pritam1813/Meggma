# Getting Started

## Installation

1.  Download the `Meggma.zip` theme file.
2.  Go to your Shopify Admin > **Online Store** > **Themes**.
3.  Click **Add theme** > **Upload zip file**.
4.  Select `Meggma.zip` and click **Upload**.

## Activation

1.  After upload, click **Customize** to set up your homepage.
2.  When ready, click **Actions** > **Publish** to make it your live theme.

## Recommended App Configuration

This theme integrates seamlessly with standard Shopify apps. No special configuration is usually required.

## First Steps

- Go to **Theme Settings** to upload your Logo and set your brand colors.
- Set up your **Navigation** menus in the Shopify Admin.
- Customize the **Header** and **Footer** sections.
