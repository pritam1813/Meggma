# Theme Settings

Access these global settings via the **Theme Settings** icon (gear) in the Theme Editor.

## Colors
Define your brand's color palette.
- **Primary**: Main button and accent color.
- **Secondary**: Secondary actions and highlights.
- **Background**: Main site background.
- **Text**: Primary text color.

## Typography
Set the fonts for your store.
- **Headings**: Font for titles and headers.
- **Body**: Font for main content text.

## Layout
- **Page Width**: Maximum width of the content container (e.g., 1200px).
- **Spacing**: Global control for section vertical spacing.

## Social Media
Add links to your social profiles. Icons will automatically appear in the footer.
- Twitter, Facebook, Pinterest, Instagram, TikTok, YouTube.

## Checkout
Customize the look of your checkout page.
- **Banner**: Background image for the order summary.
- **Logo**: Custom logo for checkout.
